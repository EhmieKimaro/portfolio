Portfolio Details

The game was designed by me to stimulate users into finding words witin other words. The game app is designed to provide fun and engaging word challenges for users of all ages. The goal was to create an intuitive, visually appealing app that combines learning with entertainment.

Design Process

Through user research, I identified the need for a simple yet rewarding word game. Wireframes and prototypes were developed to map key features like drag-and-drop word formation, daily puzzles, and leaderboards. User feedback during testing ensured the design was easy to navigate and engaging.

Features

1. Timed word challenges with varying difficulty.

2. Daily puzzles to keep users motivated.

3. Leaderboards for friendly competition.

4. Progress tracking to monitor improvement.

The Word Game App is a fun way to improve vocabulary while staying entertained.
